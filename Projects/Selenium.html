AutoSales Analysis Arkansas

This project involved analyzing automotive sales data in Arkansas using Python and Power BI. The goal was to uncover trends and provide actionable insights for dealerships.

Tools Used

  • Python (Pandas, Matplotlib)
  • Power BI
  • Excel

Project Overview

- Cleaned and preprocessed raw data using Python.
- Created interactive dashboards in Power BI for trend visualization.
- Provided recommendations for improving sales in low-performing areas.
